# 🧩 Maze Solver using A* Algorithm

##  Overview
This project implements the A* (A-Star) Search Algorithm to find the shortest path in a maze. The maze is represented as a grid where some cells are blocked, and the algorithm intelligently finds the optimal route from the start point to the goal.

---

##  What is A* Algorithm?
A* is a pathfinding algorithm widely used in Artificial Intelligence. It combines:
- Actual cost from start node (g(n))
- Estimated cost to goal (h(n)) using a heuristic

Formula:
f(n) = g(n) + h(n)

---

##  Features
- ✅ Finds the shortest path in a maze
- ✅ Uses Manhattan Distance heuristic
- ✅ Handles blocked paths (walls)
- ✅ Simple console-based visualization
- ✅ Beginner-friendly implementation

---

## Tech Stack
- Python
- Heap Queue (Priority Queue)

---

##  Maze Representation
- `0` → Free path  
- `1` → Wall  
- `S` → Start  
- `G` → Goal
